# Adding Colby Campus Image

To add a Colby College campus image to the login/signup pages:

1. Find or download a high-quality image of Colby College campus
2. Save it as `colby-campus.jpg` in the `public/` folder
3. The image will automatically appear on the left side of the login and signup pages

**Recommended image specifications:**
- Format: JPG or PNG
- Dimensions: At least 1200x800 pixels (will be cropped/covered)
- File size: Under 2MB for faster loading
- Content: Colby College campus, buildings, or scenic view

**Alternative:**
If you don't have an image, the pages will show a solid navy blue background (Colby blue) with the text overlay.
